# Tailwind CSS - Nike Website

Dive into the world of tailwindcss and build a Nike website.

## Demo:

## Install npm Packages:

#### `npm install -D tailwindcss postcss autoprefixer`

## Run App Locally:

#### `npm run dev`

## Deploy App:

#### `npm run build`

#### `npm run preview`
